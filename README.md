# To-Do-List

Overview

This project is a basic task management application that allows users to create, view, and manage tasks directly from the terminal.

This project is a basic task management system that runs in the terminal.
It enables users to manage a list of tasks interactively using a menu-driven interface.

The application stores tasks temporarily in memory using a Python list.

Features
Add new tasks

View all current tasks

Remove tasks by selecting their number

Clear and simple command-line interface

Function-based structure


How It Works
The program:

Initializes an empty list to store tasks

Displays a menu with available options

Accepts user input to select an action

Updates the task list based on the selected option
